Options become the full control room

The popup intentionally stops trying to be both a status dashboard and a configuration form. The options page carries setup, model policy, and advanced details. That split makes the popup calmer and faster.

What changes

One dominant control, one short state sentence, deferred detail drawer. Settings remain available, but not as the center of gravity.

Why it helps

Most users only need confidence that the extension is on. This removes form-thinking from the popup and lowers scan time.
